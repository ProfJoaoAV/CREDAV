1. Palavras Simples vs Complexas

Palavras Simples

Não provêm de nenhuma outra palavra. Funcionam como a semente (ou base) para criar palavras novas.

Ex: cereja, atar, rolha, sapato

Palavras Complexas

Provêm de outra palavra. Formadas por mais de um constituinte (Derivação ou Composição).

Ex: cerejeira, desatar, saca-rolhas

Radical: Contém o significado básico da palavra.
Exemplo: cerej-eira

2. Derivação (Prefixo + Base + Sufixo)

Juntar a uma forma de base (radical ou palavra) um prefixo (antes) ou um sufixo (depois).

Prefixos

  • i-, in-, im-, ir- (negação) ➔ ilegal, inativo, impossível, irreal
  • des- (ação contrária) ➔ destapar
  • pré- (anterior) ➔ pré-natal
  • pós- (posterior) ➔ pós-cirurgia
  • re- (repetição) ➔ refazer
  • sobre- / sub- (superior/inferior) ➔ sobrecarga, subdelegado
🔥 Poder NeGativo: Os 4 "I"s (I, In, Im, Ir) e o "Des" destroem o significado! (legal -> ilegal).

Sufixos

  • -ão, -zão, -zarrão, -aço (aumento) ➔ carrão, canzarrão, ricaço
  • -inho, -ito, -ote (diminuição) ➔ carrinho, filhote
  • -agem (ação/resultado) ➔ secagem
  • -eiro, -ista (profissão) ➔ carteiro, artista
  • -aria (local) ➔ frutaria
  • -ês, -eiro, -ense (naturalidade) ➔ português, brasileiro
  • -ada, -al (conjunto) ➔ ninhada, pinhal

3. Composição

Juntar 2 ou mais bases
Palavra + Palavra
Duas palavras que já existem sozinhas.
saca-rolhas
Radical + Palavra
Um radical junta-se a uma palavra.
hidroavião
Radical + Radical
Dois radicais (grego/latim) juntos.
cronómetro
